牌陣及使用步驟說明





新手進行塔羅牌時必了解４步驟


多數人對於自己的心中的問題總是不夠清楚，想獲得答案是自己的還是對方的心情、給予自己執行上的建議還是單純想知道做完這件事後的運勢。
當我們用模糊不清楚的方式進行問題，自然也只能得到模糊的答案，進行塔羅占卜時，非常需要清楚而直白的問題。
尤其塔羅牌占卜並不是單純進行吉凶預兆，而是給予我們對於問題本身提供不同的見解與思考方式的對話，因此理解自己想問的問題非常重要。
．好的問題：「該怎麼做才能找到彼此契合的戀人？」
．不建議的問題：「為什麼我沒有情人？」
問問題的方式請以自己「想怎麼做？」的想法，搭配「該如何做才能加以實現」這樣的形式對策，才能得到較明確而詳細的回饋。

決定完問題後，可以透過牌陣來得出自己想要的解答，牌陣可以因應問題的方式以單張牌／二選一／時間之流等牌陣來進行占卜。
建議在占卜前，先抽一張「現在的自己處於何者狀況？」的牌，給予此刻自己詮釋、了解現階段自己的心理狀態。

塔羅牌占卜是透過偶然抽出的牌進行解讀，所以每次進行詢問時都需要重新洗牌、隨機將牌卡排列才能得到相對應的解答。
而透過轉動牌卡的過程，也能讓自己重新集中精神，並找出心中的那張解答卡。

想著自己的問題後，再翻開牌卡，詳細解讀上頭顯示的文字與提醒，這是你自身潛意識與牌卡解答相連的象徵。
塔羅牌占卜的準確度會隨著占卜的次數而提升，在解讀的過程中也會因為次數多而更加得心應手；因此很也很建議在每次占卜完後都做簡單的筆記、記錄此次的問題與解答。




常見牌陣

✦ 單張牌：以一問一答的方式引出答案
多用於沒有時間限制，想直截了當得到解答時使用；除此之外也能用於日常對於未來的運勢測驗，得出簡單的預測。 建議的問題：想掌握目前的狀況、想知道自己／對方的心情、想得到接下來的發展、想知道接下來該如何執行、想得到針對式的單一問題解答。 可嘗試用這些方式問問題：今天該注意什麼事情？那個人的態度突然變冷淡，對方現在的想法是？


✦ 三張牌｜時間之流：
過去➊ 》現在➋ 》未來➌ ; 身➊ 》心➋ 》靈➌ ; 原因➊ 》目前結果➋ 》建言➌ ; 建議➊ 》建議➋ 》建議➌


✦ 三張牌｜聖三角牌陣：
我以為的狀態➊ 》真實的狀態➋ 》建言➌


✦ 四張牌｜直指核心：
問題核心➊ 》障礙➋ 》對策➌ 》優勢➍


✦ 五張牌｜二選一＆可能的結果：
選項A的狀態➊ 》選項B的狀態➋ 》選項A可能的結果➌ 》選項B可能的結果➍ 》當事人的狀態➎


✦ 五張牌｜感情萬用牌陣：
我的狀態➊ 》本人對我的感情態度➋ 》對方的狀態➌ 》對方對我的感情態度➍ 》可能的結果➎

問對問題＆選對牌陣的重要性:

  • 第一步 : 塔羅抽牌最重要的就是問對問題，不然會難易解釋
  • 第二步 : 更重要的就是「選對牌陣」

可以自創牌陣嗎？

  • 可以
  • 先將塔羅牌握在心輪，並跟塔羅牌說「請我的高我為我回答問題，第1張牌代表“問題的根源”、2是“我的障礙點”、3是“我能做些什麼”、4是“我有什麼資源可用”…」
